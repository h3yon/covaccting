## Information
백신 종류별 후기와 백신 정보를 확인하며<br/>
접종 날짜도 알려주는 서비스<br/>
-> 후기를 공유하고 작성하며, 모두가 안전하게 백신 접종을 마치는 날까지 이 서비스는 계속됩니다!<br/>
